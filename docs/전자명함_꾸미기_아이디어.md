# 전자명함 랜딩 꾸미기 아이디어

## 이미 적용한 것
- 스크롤 시 섹션 페이드인 (살짝 위에서 등장)
- 카드 호버 시 살짝 떠오르는 느낌 + 청록 그림자
- 이름 글자 간격 강조 (tracking)
- 구분선·글로우 라인 살짝 반짝이는 애니메이션
- 섹션 사이 그라데이션 구분선

## 더 하고 싶을 때 추천

### 시각
- **프로필 사진**: 부드러운 테두리 glow 애니메이션 (pulse)
- **배경**: 미세한 점 패턴 또는 더 정교한 회로 라인 (SVG)
- **다크 모드만**: 현재가 이미 다크라서, 포인트 색만 바꿔가며 테마 변형 (예: 청록 → 보라)

### 전자명함 느낌
- **한 장 요약 카드**: 히어로를 “명함 한 장”처럼 하나의 카드 블록으로 감싸기 (둥근 모서리 + 그림자)
- **스크롤 인디케이터**: 첫 화면에 “아래로 스크롤” 화살표 애니메이션

### 인터랙션
- **버튼**: 호버 시 살짝 커지거나 밝아지기 (이미 비슷하게 적용 가능)
- **상단 헤더**: 스크롤 시 배경 blur 또는 색 살짝 진해지기 (sticky)

### 콘텐츠
- **칭찬/리뷰 한 줄**: “고객 분들이 이런 점이 좋다고 해요” 한 문장
- **실적 숫자**: “상담 000건” 등 (있으면)
