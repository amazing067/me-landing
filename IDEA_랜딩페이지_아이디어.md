# QR 명함 랜딩 페이지 아이디어

명함(프라임에셋 / 김성민)과 톤을 맞춘, 비즈니스·포트폴리오용 랜딩 페이지 구성안입니다.

---

## 1. 전체 전략

| 포인트 | 설명 |
|--------|------|
| **모바일 우선** | QR은 대부분 휴대폰으로 스캔 → 한 손으로 읽기 좋게, 버튼 크게 |
| **3초 안에 이해** | "누구인지 / 뭘 하는 사람인지 / 다음에 뭘 하면 되는지"가 바로 보이게 |
| **명함과 시각 통일** | 짙은 남색 + 청록/시안 회로 패턴 → 신뢰감·전문성 유지 |
| **행동 유도 명확** | 상담(카카오톡·전화), 포트폴리오 보기, 블로그 등 한 번에 보이게 |

---

## 2. 페이지/섹션 구성 제안

### 1페이지: 히어로 (랜딩)

- **상단**
  - 로고: PRIME ASSET / 프라임에셋 (명함과 동일)
  - 옵션: 프로필 사진 또는 아바타 (있으면 신뢰도 ↑)
- **중앙**
  - **이름**: 김성민 (크고 굵게, 명함 스타일)
  - **한 줄 소개**:  
    예) "보험·IT·유튜브, 세 가지 전문성으로 도와드립니다"
  - **QR로 들어온 분용 안내 문구** (아래 문구 참고)
- **하단**
  - **주요 CTA 버튼**
    - 카카오톡 상담 (가장 눈에 띄게)
    - 전화 걸기
    - 포트폴리오 보기 (아래 섹션으로 스크롤 또는 별도 섹션)
  - **링크 버튼**: 블로그, GitHub, 유튜브 등 (아이콘 + 텍스트)

**QR 전용 안내 문구 예시:**

- "명함을 통해 방문해 주셔서 감사합니다. 보험·보상 상담, 웹/앱·IT 구축, 유튜브 제작 문의는 아래 버튼으로 편하게 연락 주세요."
- 또는 짧게: "명함 QR로 들어오신 분 환영합니다. 상담·문의는 카카오톡·전화로 편하게 연락 주세요."

---

### 2섹션: 나의 세 가지 전문 분야

명함의 세 가지를 그대로 가져와서, 아이콘 + 짧은 설명으로 블록 3개.

| 아이콘/키워드 | 제목 | 한 줄 설명 |
|---------------|------|------------|
| 방패/체크 | 보험·보상 전문 컨설턴트 | 보험 설계·보상 컨설팅 |
| 모니터/코드 | 웹/앱 및 IT 시스템 구축 | 기업용 웹·앱·시스템 개발 |
| 필름/카메라 | 유튜브 기획·촬영·편집 | 채널 기획부터 편집까지 |

디자인: 명함처럼 어두운 배경 + 청록색 포인트, 회로 느낌의 구분선이나 장식만 넣어도 통일감 있음.

---

### 3섹션: 포트폴리오 / 만든 것들

- **치매검사**
- **어메이징사업부 시스템**
- **블로그 자동화 시스템**
- **DB 분배 시스템**
- 기타 (추가 가능)

**표현 방식 예시:**

- 카드 형태: 프로젝트명 + 한 줄 설명 + (가능하면) 썸네일/스크린샷 링크
- "더 많은 건 상담 시 말씀해 드립니다" 문구로 마무리 → 카카오톡/전화 유도

---

### 4섹션: 연락처

- 전화번호 (클릭 시 `tel:` 링크)
- 이메일 (클릭 시 `mailto:`)
- 카카오톡, 블로그, GitHub 버튼 반복 배치

푸터에 PRIME ASSET / 프라임에셋 로고 한 번 더 넣어도 좋음.

---

## 3. 디자인 방향 (명함과 맞추기)

- **색상**
  - 배경: 짙은 남색 ~ 차콜 (`#0f172a`, `#1e293b` 등)
  - 강조: 청록/시안 (`#22d3ee`, `#06b6d4` 등)
  - 텍스트: 흰색 + 연한 회색
- **장식**
  - 회로 기판 느낌의 얇은 라인, 노드(점), 각도 있는 선
  - CSS만으로 구현 가능 (gradient, border, box-shadow)
- **타이포**
  - 이름: 굵은 sans-serif
  - 본문: 가독성 좋은 sans-serif (Pretendard, Noto Sans KR 등)

---

## 4. 기술 스택 (제안)

- **Vite + React + TypeScript** → 빌드 후 정적 HTML/JS/CSS
- **스타일**: Tailwind CSS (반응형·다크 테마 빠르게 구현)
- **호스팅**: Vercel 또는 Netlify (Git 연동, 무료)
- **도메인**: me.어메이징사업부.com 연결

---

## 5. 우선순위 정리

1. **1페이지에서 이름 + 한 줄 소개 + QR 안내 문구 + 연락 버튼**까지 한 화면에
2. **모바일에서 버튼이 잘 보이고**, 카카오톡·전화 클릭이 바로 되게
3. **세 가지 전문 분야**로 "이 사람이 뭘 하는 사람인지" 3초 안에 전달
4. **포트폴리오**는 카드 몇 개로 요약 후, 상세는 상담 시 설명하는 식으로
5. **명함과 색감·분위기** 맞추기 (남색 + 청록)

원하시면 이 구조대로 **Vite + React + TypeScript + Tailwind** 프로젝트 뼈대를 만들어 드리겠습니다.  
(1페이지 히어로 + 전문 분야 + 포트폴리오 + 연락처 섹션까지 골격 코드)
