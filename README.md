# 김성민 랜딩 페이지 (QR 명함용)

명함 QR로 접속하는 비즈니스·포트폴리오 랜딩 페이지입니다.

## 실행

```bash
npm install
npm run dev    # 개발 서버 (http://localhost:5174)
npm run build  # 정적 빌드 → dist/
npm run preview # 빌드 결과 미리보기
```

## 수정할 것

- **`src/config.ts`**  
  전화번호, 카카오톡 링크, 블로그, 유튜브 URL을 실제 값으로 바꿔 주세요.

### 로고·프로필 사진 넣는 위치

아래 파일들을 **`public` 폴더**에 그대로 넣으면 됩니다.

| 넣을 파일 | 파일 이름 |
|-----------|-----------|
| 프라임에셋 로고 (가로형) | `PrimeAsset_ci_가로.png` |
| 프라임에셋 로고 (세로형) | `PrimeAsset_ci.png` |
| 어메이징사업부 로고 | `어메이징사업부로고.png` |
| 내 프로필 사진 | `200127_김성민 9003.png` |

- 가로형: 상단 헤더 왼쪽, 세로형: 하단 푸터. 어메이징사업부 로고: 상단 헤더 오른쪽.
- 프로필 파일명에 공백이 있으면 그대로 두고, config에는 경로를 `'/200127_김성민%209003.png'` 처럼 공백을 `%20`으로 넣어 두었습니다.

## 배포

- `npm run build` 후 `dist/` 폴더를 Vercel 또는 Netlify에 업로드(또는 Git 연동)하면 됩니다.
- 도메인: me.어메이징사업부.com 연결 가능.

## 구성

- **히어로**: 이름, 한 줄 소개, QR 안내 문구, 카카오톡/전화/포트폴리오 버튼
- **전문 분야**: 보험·보상 / 웹·앱·IT / 유튜브
- **포트폴리오**: 치매검사, 어메이징사업부 시스템, 블로그 자동화, DB 분배 시스템
- **연락처**: 전화, 이메일, 링크 버튼, 프라임에셋 푸터
